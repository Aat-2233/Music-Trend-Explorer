# Music-Trend-Explorer
 Music Trend Explorer is a Flask-based web application that analyzes Spotify artist data using Python, Pandas, and Matplotlib. It visualizes trends in popularity, followers, and genres, provides partial artist search, and includes a foundation for popularity prediction using machine learning.
